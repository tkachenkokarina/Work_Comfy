Програми:

Бавовна - для одягу з натуральних матеріалів: постільної білизни, спідньої білизни, рушників, сорочок, тощо

Інтенсивне прання - для сильно забрудненого одягу

Еко – для легко забрудненого одягу з економією часу та електроенергії

Штори - дбайливе прання штор та фіранок

Синтетика - для одягу із синтетичних тканин

Делікатне - для прання речей із шовку, мережива та інших делікатних тканин
