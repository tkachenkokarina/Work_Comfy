БЕЗОПАСНОСТЬ ПРЕВЫШЕ ВСЕГО

Блокировка

Варочная поверхность Miele имеет несколько помощников в сфере безопасности:

  • если на включенной поверхности нет посуды или варочная зона нагревается в течение необычайно длительного времени — она автоматически выключается;
  • индикатор остаточного тепла напоминает, что нельзя прикасаться к конфорке, пока она еще не остыла.

УМНАЯ ПОВЕРХНОСТЬ

Распознает посуду

Чтобы начать готовить, нужно просто поставить кастрюлю на конфорку. Специальный элемент управления автоматически активируется, а для эффективного использования энергии Miele определяет размер посуды.

TWINBOOSTER

Быстрый нагрев

Идеальный вариант для быстрого кипячения воды! Функция TwinBooster концентрирует мощность двух конфорок на одной, что ускоряет результат. С этой функцией также можно распределить мощность индукции отдельно для гибкого приготовления.

STOP&GO

Пауза? Легко!

Не нужно больше выключать варочную поверхность, когда нужно ненадолго отойти из кухни. Просто воспользуйтесь функцией Stop&Go, которая устанавливает минимальный уровень нагрева для рабочих поверхностей, пока вы отлучились.

БЕЗПЕКА ПОНАД УСЕ

Блокування

Варильна поверхня Miele має декілька помічників в безпековій сфері:

  • якщо на ввімкненій поверхні немає посуду або варильна зона нагрівається протягом незвично тривалого часу — вона автоматично вимикається;
  • індикатор залишкового тепла нагадує, що не можна торкатися конфорки, поки вона ще не охолола.

РОЗУМНА ПОВЕРХНЯ

Розпізнає посуд

Щоб почати готувати, вам потрібно лише поставити каструлю на конфорку. Спеціальний елемент керування активується автоматично, а для ефективного використання енергії Miele визначає розмір посуди.

TWINBOOSTER

Швидке нагрівання

Ідеальний варіант для швидкого кип’ятіння води! Функція TwinBooster сконцентровує потужність двох конфорок на одній, що пришвидшує результат. З цією функцією також можна розподілити потужність індукції окремо для гнучкого приготування.

STOP&GO

Пауза? Легко!

Не потрібно більше вимикати варильну поверхню, коли треба ненадовго відійти з кухні. Просто скористайтеся функцією Stop&Go, яка встановлює мінімальний рівень нагріву для робочих поверхонь, поки ви відлучилися.
