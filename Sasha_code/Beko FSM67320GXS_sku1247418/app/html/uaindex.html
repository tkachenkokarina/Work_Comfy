SteamShine® — очищення парою

Дозвольте гарячій парі виконати за вас усю чорнову роботу. Просто помістіть в духовку деко з невеликою кількістю води й активуйте функцію SteamShine. Уже через кілька хвилин інтенсивна пара розм’якшить залишки їжі та жиру, що пригоріли, на стінках духової шафи, після чого їх можна буде витерти кількома легкими рухами.

3D-випікання: смачно, як із пічки

Інновація для максимально простого й ефективного приготування страв у духовці. Ця функція активує всі нагрівальні елементи, тому прогрівання ведеться на всіх рівнях одразу. Тепло розподіляється по всій духовці, забезпечуючи рівномірне пропікання будь-якої кількості страв. Ця функція також використовується для одночасного приготування страв на різних рівнях.
