Технологія Duo Clean

Основна насадка для прибирання складається зі щетинної та валикової щіток. Тому нема потреби змінювати насадки, залежно від типу поверхні:

  • щетинна щітка глибоко очистить всі види килимів із низьким, середнім та високим ворсом
  • щітка-валик допоможе ретельно прибрати пил, дрібне та грубе сміття
  • обидві щітки є моторизованими, а отже, вони постійно обертаються під час прибирання та ще краще очищують поверхню
